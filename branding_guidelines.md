# Branding Guidelines - Origin & Iron

## Logo
**Typeface:** Inter (Bold/700)
**Tracking:** 3px
**Color:** White (#FFFFFF) or Foreground (#0B0B0B) depending on background.

## Color Palette

### Primary
- **Copper**
  - Hex: `#B87333`
  - Usage: Accents, Buttons, Highlights, Subtitles

### Neutrals
- **Background (Dark)**
  - Hex: `#0A0A0A`
  - Usage: Main background
- **Card Background**
  - Hex: `#111111`
  - Usage: Product cards, Menu sections
- **Border**
  - Hex: `#333333`
  - Usage: Grid lines, Dividers

### Text
- **Foreground (White)**
  - Hex: `#FFFFFF`
  - Usage: Headings, Body text on dark
- **Muted Foreground**
  - Hex: `#A0A0A0`
  - Usage: Secondary text, descriptions, footer links

## Typography

### Primary Font: Inter
- **Weights:**
  - Light (300): Hero titles, Menu headers
  - Regular (400): Body text, Section titles
  - Medium (500): Buttons, Product names
  - Bold (600/700): Prices, Logo
  
### Hierarchy
- **H1 (Hero):** 64px, Light (300), Tight-leading
- **H2 (Section):** 32px, Regular (400)
- **H3 (Menu):** 24px, Light (300)
- **Eyebrow:** 14px, Uppercase, Wide-tracking (4px)

## Spacing & Layout
- **Grid:** 12-column logic (though implemented with CSS Grid).
- **Container:** Max-width 1200px.
- **Padding:** Generous whitespace (120px section padding) to create a premium feel.
